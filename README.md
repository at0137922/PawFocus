# PawFocus

A cross-platform Elanco-themed Pomodoro study app built with React Native (Expo).

## Features
- Pomodoro-style timer with breaks
- Background music player
- Flashcard notifications
- AI-powered chat assistant

---

## Project Structure

```
/src
  /components   # Reusable UI pieces (Timer, Flashcard, Chat, Music)
  /screens      # Screens for navigation (Home, Timer, Flashcards, Chat, Music)
  /services     # API calls and helpers
  /utils        # Utility functions (timer logic, etc.)
App.js          # App entry and navigation
```

## Setup
1. Install Expo: `npm install -g expo-cli` or use `npx expo ...`
2. Run app: `npx expo start`

---

For more details on feature development, check `src/screens/` and `src/components/` folders.
